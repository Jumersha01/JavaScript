vi.local storage vs session storage
browser can even store the data using local storage or session storage.
local storage is permenetn stays even after browser closes
session storage is temparary and expires once the browser closes

local storage can be sharesd with all the tabs
session storage stays only for one tab

local or session storage uses similar methods such as 
localStorage.setItem(itemName,value)---always store the value as string
localStorage.getItem(itemName)

localStorage.removeItem(itemName)
